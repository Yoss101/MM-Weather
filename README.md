# MM-Weather

This is a weather script designed for use with [meshmonitor](https://meshmonitor.org/)

## Description

This script is used to pull weather data from api.weather.gov using the requesting nodes lat/long. If the requesting node DOES NOT have any gps data; the location of the meshmonitor instance will be used instead.
